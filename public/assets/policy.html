Политика обработки персональных данных

ФЗ-152

Общие положения

Настоящая Политика определяет порядок обработки персональных данных пользователей веб-сайта «Учебные мини-игры» (далее — Сайт). Сайт создан в рамках учебного проекта «От формул к практике через игру» учениками 8 класса Московской международной школы.

Какие данные собираются

При использовании Сайта обрабатываются следующие персональные данные:

  • Номер телефона — для авторизации через Telegram
  • Имя пользователя — для персонализации интерфейса
  • Игровой прогресс — результаты прохождения игр, достижения, сохранения

Цели обработки данных

Персональные данные обрабатываются исключительно для:

  • Идентификации пользователя при входе на Сайт
  • Сохранения игрового прогресса между сессиями
  • Отображения персонализированного приветствия
  • Анализа эффективности обучающих игр в рамках учебного исследования

Хранение данных

Все персональные данные хранятся на серверах Яндекс Cloud, расположенных на территории Российской Федерации, что соответствует требованиям Федерального закона №152-ФЗ «О персональных данных».

Важно: Ваши данные не передаются третьим лицам, не используются в рекламных целях и не продаются. Данные используются только для функционирования Сайта и учебных целей проекта.

Срок хранения

Персональные данные хранятся в течение срока существования учебного проекта и могут быть удалены по запросу пользователя. После завершения проекта все данные будут обезличены или удалены.

Права пользователя

Вы имеете право:

  • Получить информацию о своих персональных данных
  • Потребовать исправления неточных данных
  • Отозвать согласие на обработку и потребовать удаления данных
  • Выйти из системы и прекратить использование Сайта

Для реализации своих прав вы можете выйти из личного кабинета (кнопка «Выйти») или связаться с авторами проекта.

Согласие на обработку

Вводя номер телефона и нажимая кнопку «Получить код», вы даёте согласие на обработку персональных данных в соответствии с настоящей Политикой. Согласие может быть отозвано в любой момент путём прекращения использования Сайта и выхода из системы.

Контакты

Проект выполнен учениками 8 класса Московской международной школы в рамках конкурса проектных работ «Меркурий». По вопросам обработки персональных данных обращайтесь к руководителю проекта через администрацию школы.
